🎵 Background Music Folder
========================

Put your background music file here!

Supported formats:
- MP3 (.mp3) - Recommended
- WAV (.wav)
- OGG (.ogg)

Instructions:
1. Add your music file to this folder
2. Rename it to: background-music.mp3 (or .wav, .ogg)
3. The website will automatically play it when users interact with the page

Note: Due to browser autoplay policies, music will only start playing 
after the user clicks a button (Message or Album).

Recommended: Use a soft, romantic instrumental track
File size: Keep under 10MB for faster loading
